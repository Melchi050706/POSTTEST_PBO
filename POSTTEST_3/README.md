LAPORAN POSTTEST 3

Sistem Manajemen Karakter Wuthering Waves

Konsep yang digunakan:
- Inheritance
- Hierarchical Inheritance

Penjelasan:

Program ini menggunakan konsep inheritance dimana class Karakter sebagai parent class.

Kemudian terdapat dua subclass yaitu:
1. DPS
2. Support

Kedua class tersebut merupakan turunan dari class Karakter (relasi is-a).

DPS adalah Karakter
Support adalah Karakter

Inheritance diterapkan menggunakan keyword extends.

Selain itu digunakan keyword super untuk mengambil constructor dari parent class.

Fitur:
- Tambah data DPS
- Tambah data Support
- Lihat data
- Hapus data
