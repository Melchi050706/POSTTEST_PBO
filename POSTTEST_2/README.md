2. Rincian Program

Program ini merupakan aplikasi berbasis bahasa Java yang digunakan untuk mengelola data karakter dari game Wuthering Waves.  

Pada Posttest 2 ini, program dikembangkan dengan menerapkan konsep Object Oriented Programming yaitu Encapsulation.  

Encapsulation merupakan konsep yang menyembunyikan data di dalam class sehingga tidak dapat diakses secara langsung dari luar class, melainkan melalui method getter dan setter.

Program menggunakan ArrayList untuk menyimpan objek karakter sehingga data dapat ditambah, diubah, ditampilkan, dan dihapus secara dinamis.

3. Fitur Utama

1. Tambah Karakter

2. Lihat Karakter

3. Update Karakter  
Mengubah informasi karakter yang sudah ada.

4. Hapus Karakter  
Menghapus data karakter dari sistem.

5. Keluar Program  
Mengakhiri program.

4. Konsep Encapsulation

Pada program ini diterapkan konsep encapsulation dengan cara:

1. Atribut pada class Karakter dibuat private sehingga tidak dapat diakses langsung dari luar class.
2. Untuk membaca nilai atribut digunakan method getter.
3. Untuk mengubah nilai atribut digunakan method setter.

Access modifier yang digunakan dalam program ini yaitu:

- private
- public

5. Class Pada Program

Program ini menggunakan dua class yaitu:

Class Karakter  
Class ini digunakan untuk menyimpan data karakter seperti nama, elemen, dan level.

Class Main  
Class ini berisi program utama yang menjalankan menu CRUD serta mengatur alur program.
